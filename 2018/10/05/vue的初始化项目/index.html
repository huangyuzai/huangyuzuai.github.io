或者看我这篇博客 –> 使用Github Pages + Hexo 搭建个人博客

安装 Node.js

安装也很简单，自己去百度。
或者，还是看上面那篇博客
下载地址在这 –> Node.js官网

安装 cnpm(可有可无)

如果嫌弃 npm 慢的话，可以安装 cnpm 。
当然也是可以不安装的，不安装的话跳过这步就可以啦

有两个版本可以选择

安装完成以后可以用 cnpm -v 查看一下版本号是否安装成功
